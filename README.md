# Qiongjing Cosmora

> AI-driven personalized resource generation and learning multi-agent system

## Overview
Qiongjing Cosmora is an AI-powered smart education platform integrating learning profiling, knowledge quizzes, 1-on-1 AI tutoring, document reading assistance, and resource generation.

## Tech Stack
- Frontend: Vue 2 + Element UI + ECharts
- Backend: Spring Boot + MyBatis + MySQL
- AI: iFLYTEK Spark / HiDream / ChatDoc / Voice Services
- Live2D: Yili AI Tutor Character

## Features
- Learning Closed Loop
- Knowledge Quiz (30 sets, ~270 questions)
- AI 1-on-1 Tutoring
- Document Reading & Analysis
- Resource Generation (Image/Chart/PPT)
