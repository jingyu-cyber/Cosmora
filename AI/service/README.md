# AI Interview Service

Flask-based AI interview service with video/voice/face emotion analysis.

## Features
- AI-powered interview simulation
- Video recording & voice recognition
- Face emotion analysis
- Multi-turn follow-up questions
- 7-dimension scoring & reports

## Tech
- Flask + SQLite
- Ollama (local LLM)
- XFYUN Voice API
- OpenCV + FER (face emotion)
